The important files are:

Jack_the_Nipper_2 - B0.txt - The disassembly
Jack_the_Nipper_2 - B0.PRG - The binary file on which the disassembly is based
Jack_the_Nipper_2.tap - Game as a TAP file (My original source)

Various CCS64 state files:

freeze - JTN2 - CHEAT.c64
freeze - JTN2 - Patched.c64 - A patched version which watches two memory locations
                              (Patch.txt - The disassembly of the patch)
freeze - JTN2.c64
freeze0.c64
freeze1.c64

There is also a "Jack_the_Nipper_2 - B5.PRG" file which is the same as "Jack_the_Nipper_2 - B0.PRG",
but saved with a different BANK setting in CCS64. This file is not important.


How to run:
Load state "freeze - JTN2.c64" or "freeze - JTN2 - Patched.c64" into CCS64
